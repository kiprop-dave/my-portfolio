ESP32

Door Access using RFID reader.

In this project, we were trying to create a door access that uses Radio Frequency identification (RFID) reader. In this setup, we used the ESP32 microcontroller to read the RFID tags and then compare the tag number to the stored tag numbers. If the tag number matches, the solenoid lock is engaged and the door is opened. If the tag number does not match, the solenoid lock is disengaged and the door is closed. We used an n-channel MOSFET as a switch to control the solenoid lock. We used a buck converter to step down the 12V DC supply to 9V which is used by the solenoid lock. You can see it working here.
